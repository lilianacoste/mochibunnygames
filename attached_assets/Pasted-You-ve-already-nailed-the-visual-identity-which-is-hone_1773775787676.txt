# 🌸 CORE PAGES (Must-Have)

## 🏠 1. Homepage (Your Hook)

This is where people decide:
👉 “this looks cute”
👉 “…wait why does this feel off?”

**Add:**

* 🎮 Featured game (Project Lottie.exe)
* Short 1–2 line hook:

  > “A cozy desktop… that’s watching you back.”
* Play / Wishlist buttons
* 1 unsettling GIF (subtle glitch 👀)

💡 *Important:* Don’t explain too much. Mystery = clicks.

---

## 🎮 2. Games Page

Even if you only have ONE game, still make this page.

**Why?**
It makes you look like a *studio*, not a project.

**Include:**

* Game card(s)
* Status tags:

  * “In Development”
  * “Demo Available”
* Tiny tags:

  * Psychological Horror
  * Analog Horror
  * Puzzle
